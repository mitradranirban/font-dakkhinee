Dakkhinee - A Bangla font inspired by South Indian scripts

Dakkhinee is an attempt to visualise how Bengali script would have evolved had it been a South Indian scripts like Telegu or Kannadda.

Each Indic script has evolved in it own way from Brahmi and Kharosti depending upon materials used in writing

About

The font is being created by Dr Anirban Mitra, an amateure typographer involved in creating one of the earliest Open Source OpenType Bangla font.

The Latin glyphs are also given a Kannada like look.

The Glyph design is inspired by Lohit Kannada Font

Building

Fonts are built automatically by GitHub Actions. We will release the beta when ready


If you want to build and test fonts manually on your own computer:

  • Download the source code from our github repository.

  • Create a Python virtual enviroment in the root of the repo:

    python3 -m venv venv

  • Activate venv:

    source venv/bin/activate

  • Install general dependencies:

    pip install --upgrade pip

    pip install -r requirements.txt

    pip install -r requirements-test.txt

  • make build will produce font files.
  • make test will run FontBakery's quality assurance tests.
  • make proof will generate HTML proof files.

The QA tests and proof files are also available automatically via GitHub Actions
